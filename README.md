# University Admin

## To Downlaod 
```bash
  git clone https://github.com/GabrielJospin/dev_Sist_university_admin.git
  cd university_admin
```

## To Build

```bash
  ./gradlew build
```

## To Run
```
  ./gradlew bootRun
```
And after that, open the localhost in port 8080: [localhost:8080/](localhost:8080/)
